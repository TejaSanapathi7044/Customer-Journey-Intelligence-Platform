# Customer Journey Intelligence Platform

## Project Overview

Built an end-to-end cloud analytics pipeline using AWS S3, AWS Glue, AWS RDS (MySQL), SQL, Python, and Power BI to analyze customer journeys, revenue trends, delivery performance, and customer behavior.

## Architecture

S3 → AWS Glue → AWS RDS (MySQL) → SQL Analytics → Power BI Dashboard

## Technologies Used

- Python
- Pandas
- SQL
- AWS S3
- AWS Glue
- AWS RDS (MySQL)
- Power BI

## Key Features

- Customer Journey Analysis
- Revenue Analytics
- Delivery Performance Tracking
- Cohort Analysis
- Review Analysis
- Dashboard Automation
- Incremental Data Loading
